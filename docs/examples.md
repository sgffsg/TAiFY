# Примеры программ

## 1. SumNumbers

```
(ПОЯСНИТЕЛЬНАЯ-БРИГАДА: Складывает два числа)

ПРОКРАСТИНИРУЕМ ПОГНАЛИ() 
ПОЕХАЛИ
    ПОЛТОРАШКА первоеЧисло = 0.0;
    ПОЛТОРАШКА второеЧисло = 0.0;
    
    ВБРОС(первоеЧисло, второеЧисло);
    ПОЛТОРАШКА сумма = первоеЧисло + второеЧисло;
    
    ВЫБРОС(сумма);
ФИНАЛОЧКА
```

## 2. CircleSquare

```
(ПОЯСНИТЕЛЬНАЯ-БРИГАДА: Вычисляет площадь круга)

ПРОКРАСТИНИРУЕМ ПОГНАЛИ() 
ПОЕХАЛИ
    ПОЛТОРАШКА радиус = 0.0;

    ВБРОС(радиус);
   
    ПОЛТОРАШКА площадьКруга = ПИ * радиус * радиус;

    ВЫБРОС(площадьКруга);

```

## 3. MilesToKm

```
(ПОЯСНИТЕЛЬНАЯ-БРИГАДА: Глобальная константа для перевода)
БАЗА ПОЛТОРАШКА КОЭФФИЦИЕНТ = 1.60934;

ПРОКРАСТИНИРУЕМ ПОГНАЛИ() ПОЕХАЛИ
    ПОЛТОРАШКА расстояниеМили = 0.0;

    ВБРОС(расстояниеМили);

    ПОЛТОРАШКА расстояниеКм = расстояниеМили * КОЭФФИЦИЕНТ;

    ВЫБРОС(расстояниеКм);
ФИНАЛОЧКА
```
