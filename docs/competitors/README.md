# Сравнение языков программирования C++ и Pascal

## 1. Парадигма

### Pascal следует исключительно императивной парадигме:
- Программа как последовательность команд, изменяющих состояние
- Акцент на процедурах и функциях, оперирующих данными
- Жесткая структура: данные и код строго разделены

### C++ - мультипарадигменный язык программирования:
- **Императивная** - процедурное программирование
- **Объектно-ориентированная** - классы, наследование, инкапсуляция
- **Обобщенное программирование** - шаблоны, метапрограммирование
- **Функциональная** - лямбды, функции высшего порядка
- **Абстракции данных** - перегрузка операторов, пользовательские типы

## 2. Структурное программирование

### Pascal
- **Циклы:** `for var := start to/downto end do`, `while condition do`, `repeat until condition`
- **Ветвления:** `if then else`, `case of`
- **Блоки:** `begin end`

### C++
- **Циклы:** `for (init; cond; inc)`, `while (cond)`, `do while (cond)`
- **Ветвления:** `if else`, `switch case`
- **Блоки:** `{ }`

## 3. Объявление типов

### Pascal
```
var x: integer;
function f(a: integer): boolean;
```

### C++
```cpp
int x;
bool f(int a);
```

## 4. Базовые типы данных

### Pascal
- `integer`
- `real`
- `char`
- `boolean`
- `string`

### C++
- `int`
- `float`/`double`
- `char`
- `bool`
- `char[]`
- `std::string`

## 5. Пользовательские функции
**Pascal:**
- Разделение `function` (возвращает значение) и `procedure` (не возвращает)
- Рекурсия поддерживается
- Параметры: по значению, по ссылке (`var`)

**C++:**
- Единый механизм функций
- Рекурсия поддерживается
- Параметры: по значению, по ссылке (`&`), по указателю (`*`)

## 6. Пользовательские структуры
**Pascal:** `record ... end`
```
type Point = record
    x, y: integer;
end;
```

**C++:** `struct ... {}`
```
struct Point {
    int x, y;
};
```

## 7. Строки
**Pascal:** Встроенный тип `string` (фиксированной длины), операции +, =, <>
**C++:** Массивы `char[]` с нуль-терминатором, библиотечные функции

## 8. Массивы
**Pascal:** Статические, с явными границами
```
var arr: array[1..10] of integer;
```

**C++:** Статические/динамические, нумерация с 0
```
int arr[10];
```

### 9. Управление памятью
**Pascal:** Автоматическое, `new/dispose` для указателей
**C++:** Ручное: `new/delete`, автоматическое для стека

### 10. Обработка ошибок
**Pascal:** Исключения через `try/except`, `try/finally`
**C++:** `try/catch` + коды возврата

### 11. Операторы
**Арифметические:**
- Pascal: `+ - * / div mod`
- C++: `+ - * / %`

**Логические:**
- Pascal: `and or not`
- C++: `&& || !`

**Сравнения:**
- Pascal: `= <> < > <= >=`
- C++: `== != < > <= >=`

**Присваивание:**
- Pascal: `:=`
- C++: `=`
### Ключевые выводы для реализации фронтенда:
1. **Лексический анализ:** Разные ключевые слова и операторы
2. **Синтаксический анализ:**
    - Разная грамматика объявлений (тип до/после идентификатора)
    - Разные правила областей видимости
3. **Семантический анализ:**
    - Разная система типов
    - Разные правила приведения типов
4. **Генерация промежуточного кода:** Разные модели памяти и вызова функций

Pascal проще для учебного компилятора благодаря строгой структуре и меньшему количеством конструкций. C++ требует обработки более сложных выражений и указателей.
