# Различие лексической структуры C++ и Pascal

## 1. Блоки кода

* **Pascal**: Логические блоки кода заключаются между ключевыми словами **`begin`** и **`end`**.
* **C++**: Для выделения блоков кода используются фигурные скобки **`{ }`**.

## 2. Ветвление (условные операторы)

| Конструкция                 | Pascal                                          | C++                                       |
|:----------------------------|:------------------------------------------------|:------------------------------------------|
| **Простое условие**         | `if <условие> then <действие>;`                 | `if (условие) { ... }`                    |
| **Условие с альтернативой** | `if <условие> then <действие> else <действие>;` | `if (условие) { ... } else { ... }`       |
| **Множественный выбор**     | `if ... then ... else if ... then ...;`         | `if (...) { ... } else if (...) { ... }`  |
| **Выбор из вариантов**      | `case <переменная> of ... end;`                 | `switch (переменная) { case ... break; }` |

*Ключевое отличие*: В Pascal после `then` и `else` может стоять только одна инструкция. Если нужно выполнить несколько,
их необходимо обернуть в блок `begin...end`. В C++ фигурные скобки `{...}` обязательны для блоков, но могут быть
опущены, если в ветке всего одна инструкция.

## 3. Циклы

| Тип цикла          | Pascal                    | C++                                     | Ключевое различие                                                                                               |
|:-------------------|:--------------------------|:----------------------------------------|:----------------------------------------------------------------------------------------------------------------|
| **Со счётчиком**   | `for i := 1 to 10 do ...` | `for (int i = 1; i <= 10; i++) { ... }` | В C++ можно задать любой шаг и условие; в Pascal шаг всегда равен 1 (`to` или `downto`).                        |
| **С предусловием** | `while x > 0 do ...`      | `while (x > 0) { ... }`                 | Синтаксис очень похож, разница в скобках для условия и оформлении блока кода.                                   |
| **С постусловием** | `repeat ... until x > 0;` | `do { ... } while (x <= 0);`            | **Логика условия противоположна**: Pascal повторяет цикл, пока условие **ложно**, а C++ — пока оно **истинно**. |

## 4. Средства для ввода/вывода

| Действие                                     | Pascal               | C++                              |
|:---------------------------------------------|:---------------------|:---------------------------------|
| **Вывод без переноса строки**                | `write(...)`         | `std::cout << ...`               |
| **Вывод с переносом строки**                 | `writeln(...)`       | `std::cout << ... << std::endl;` |
| **Чтение с клавиатуры**                      | `read(переменная)`   | `-`                              |
| **Чтение с клавиатуры (с переносом строки)** | `readln(переменная)` | `std::cin >> переменная;`        |

## 5. Пользовательские функции

| Аспект                              | Pascal                   | C++                |
|:------------------------------------|:-------------------------|:-------------------|
| **Процедурая**                      | `procedure Имя(...)`     | `void Имя(...)`    |
| **Функция**                         | `function Имя(...): Тип` | `тип Имя(...)`     |
| **Передача параметров по значению** | `(имя: Тип; ...)`        | `(Тип имя, ...)`   |
| **Передача параметров по ссылке**   | `(var имя: Тип; ...)`    | `(Тип& имя, ...)`  |
| **Возврат значения**                | `Result := значение;`    | `return значение;` |

*Ключевое отличие*: Pascal строго разделяет **процедуры** (которые не возвращают значение) и **функции**. В C++ всё
является **функциями**, а те, что ничего не возвращают, имеют тип `void`. Также в Pascal результат функции требуется
записать в переменную Result. В c++ достаточно же просто вызвать return

## 6. Встроенные типы данных

| Тип данных                   | Pascal               | C++                    |
|:-----------------------------|:---------------------|:-----------------------|
| **Целые числа**              | `integer`, `longint` | `int`, `long`, `short` |
| **Числа с плавающей точкой** | `real`, `double`     | `float`, `double`      |
| **Строки**                   | `string`             | `std::string`          |
| **Объявление переменной**    | `var имя: Тип;`      | `Тип имя;`             |

*Ключевое отличие*: В Pascal все переменные объявляются в специальном блоке `var`. В C++ переменные можно объявлять в
любом месте кода до их первого использования.

## 7. Выражения с операторами

| Тип операторов     | Операция              | Pascal | C++    |
|:-------------------|:----------------------|:-------|:-------|
| **Арифметические** | Присваивание          | `:=`   | `=`    |
|                    | Целочисленное деление | `div`  | `/`    |
|                    | Остаток от деления    | `mod`  | `%`    |
| **Сравнения**      | Равно                 | `=`    | `==`   |
|                    | Не равно              | `<>`   | `!=`   |
| **Логические**     | И                     | `and`  | `&&`   |
|                    | ИЛИ                   | `or`   | `\|\|` |
|                    | НЕ                    | `not`  | `!`    |

## 8. Пользовательские составные типы данных

| Тип данных             | Pascal                           | C++                                                  |
|:-----------------------|:---------------------------------|:-----------------------------------------------------|
| **Массивы**            | `array[1..10] of integer;`       | `int myArray[10];` , `std::array<int, 10> myArray{}` |
| **Структуры (записи)** | `type TPerson = record ... end;` | `struct Person { ... };`                             |

*Ключевое отличие*: Массивы в Pascal могут иметь произвольные индексы (например, `[-5..5]`), в то время как в C++
индексация массивов всегда начинается с **0**.
