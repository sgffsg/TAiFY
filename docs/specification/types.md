# Система типов языка Vaibik

## 1. Обзор

Язык Vaibik использует строгую статическую типизацию.

* Тип переменной фиксируется при объявлении.
* Приведение типов отсутствует.
* Операции разрешены только между операндами одного и того же типа.
* Составные типы данных (массивы, структуры) отсутствуют.

## 2. Типы данных

| Тип            | Описание                                 | Значение по умолчанию | Пример            |
|:---------------|:-----------------------------------------|:----------------------|:------------------|
| **ЦИФЕРКА**    | Целое 32-битное число со знаком.         | `0`                   | `42`, `-10`       |
| **ПОЛТОРАШКА** | Вещественное число (64 бита, IEEE 754).  | `0.0`                 | `3.14`, `-0.01`   |
| **РАСКЛАД**    | Логическое значение.                     | `КРИНЖ`               | `ХАЙП`, `КРИНЖ`   |
| **ЦИТАТА**     | Строка символов (Unicode). Неизменяемая. | `""`                  | `"Привет"`, `"A"` |

*Примечание: Тип `char` отсутствует. Символ представляется как `ЦИТАТА` длиной 1.*

## 3. Семантика операторов

### 3.1. Арифметические (`+`, `-`, `*`, `/`, `%`)

Разрешены только между одинаковыми типами.

* `ЦИФЕРКА` с `ЦИФЕРКА`: Результат `ЦИФЕРКА`. Деление `/` — целочисленное.
* `ПОЛТОРАШКА` с `ПОЛТОРАШКА`: Результат `ПОЛТОРАШКА`.
    * Литералы обязаны содержать точку (`5.0`).
    * Оператор `%` (остаток) запрещен для `ПОЛТОРАШКА`.
* Смешивание типов (например, `2 + 2.5`) — Ошибка.

### 3.2. Строковые операции

Единственная допустимая операция со строками — конкатенация (`+`).

* `"А" + "Б"` → `"АБ"` (Тип `ЦИТАТА`).

### 3.3. Сравнение (`==`, `!=`, `<`, `>`, `<=`, `>=`)

Разрешено только между одинаковыми типами. Результат — `РАСКЛАД`.

* **`ЦИТАТА`**: Поддерживает лексикографическое сравнение (`"a" < "b"` → `ХАЙП`).
* **`РАСКЛАД`**: `КРИНЖ` < `ХАЙП`.

### 3.4. Логические (`И`, `ИЛИ`, `НЕ`)

Применимы только к типу `РАСКЛАД`.

## 4. Работа со строками

### 4.1. Встроенная функция длины

Язык предоставляет встроенный механизм получения длины строки.

* Синтаксис: `ДЛИНА(строка)`
* Тип возврата: `ЦИФЕРКА`
* Описание: Возвращает количество символов в строке.

### 4.2. Доступ к элементам

Квадратные скобки `[]` применяются исключительно к типу `ЦИТАТА`.

* `строка[индекс]` возвращает новую `ЦИТАТУ` (подстроку длиной 1).
* Индекс должен быть типа `ЦИФЕРКА`.
* Индексация начинается с 0.
