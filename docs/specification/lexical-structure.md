# Лексическая структура языка Vaibik

## 1. Пример кода

```Vaibik
ПОЕХАЛИ 
    ФАКТ ЦИФЕРКА число = 42;
    ЦИТАТА приветствие = "Народ!";
    
    ЕСЛИ (число > 0) ТО
        ВЫБРОС(приветствие);
    ИНАЧЕ
        ВЫБРОС("Чиселка слишком маленькая");
    ФИНАЛОЧКА
ФИНАЛОЧКА
```

## 2. Ключевые слова

* Все ключевые слова регистрозависимые

| Слово                   | Предназначение                   | Контекст использования                    |
|-------------------------|----------------------------------|-------------------------------------------|
| `ПОЕХАЛИ`               | Начало программы или блока кода  | Основной блок программы                   |
| `ФИНАЛОЧКА`             | Конец блока кода                 | Завершает блоки `ПОЕХАЛИ`, `ЕСЛИ`, `ЦИКЛ` |
| `ВОЗДУХАН`              | Объявление процедуры             | `ВОЗДУХАН имя() ПОЕХАЛИ ... ФИНАЛОЧКА`    |
| `ВБРОС`                 | Ввод значения                    | `ВБРОС(переменная)`                       |
| `ВЫБРОС`                | Вывод значения                   | `ВЫБРОС(выражение)`                       |
| `ПАЧКА`                 | Массив                           | `ПАЧКА[размер]`                           |
| `ЦИТАТА`                | Строковый тип                    | `ЦИТАТА текст = "строка"`                 |
| `БАЗА`                  | Логическое "истина"              | `РАСКЛАД флаг = БАЗА`                     |
| `КРИНЖ`                 | Логическое "ложь"                | `РАСКЛАД флаг = КРИНЖ`                    |
| `РАСКЛАД`               | Логический тип                   | `РАСКЛАД переменная = БАЗА`               |
| `ПОЛТОРАШКА`            | Число с плавающей точкой         | `ПОЛТОРАШКА х = 3.14`                     |
| `ГОЙ-ДА`                | Возврат из функции               | `ГОЙ-ДА значение`                         |
| `ЦИФЕРКА`               | Целочисленный тип                | `ЦИФЕРКА счетчик = 0`                     |
| `ПШИК`                  | Null-значение                    | `переменная = ПШИК`                       |
| `ГОП-СТОП`              | Прерывание цикла                 | `ГОП-СТОП`                                |
| `ФАКТ`                  | Константа                        | `ФАКТ значение = 10`                      |
| `ЕСЛИ`                  | Условный оператор                | `ЕСЛИ (условие) ТО ...`                   |
| `ТО`                    | Часть условного оператора        | `ЕСЛИ (условие) ТО ...`                   |
| `ИНАЧЕ`                 | Альтернативная ветка             | `ЕСЛИ ... ИНАЧЕ ...`                      |
| `ЦИКЛ`                  | Объявление цикла                 | `ЦИКЛ (инициализация; условие; шаг)`      |
| `И`                     | Логический оператор И            | `условие1 И условие2`                     |
| `ИЛИ`                   | Логический оператор ИЛИ          | `условие1 ИЛИ условие2`                   |
| `НЕ`                    | Логический оператор НЕ           | `НЕ условие`                              |
| `ПОЯСНИТЕЛЬНАЯ-БРИГАДА` | Начало комментария               | `(ПОЯСНИТЕЛЬНАЯ БРИГАДА: текст)`          |
| `ФИНАЛОЧКА-КОММЕНТАРИЯ` | Конец многострочного комментария | `... ФИНАЛОЧКА КОММЕНТАРИЯ)`              |

## 3. Идентификаторы

Идентификаторы в `Vaibik` должны начинаться с буквы русского алфавита или символа подчеркивания (`_`), после которого
могут следовать буквы русского алфавита, цифры или подчеркивания. Кроме этого идентификаторы регистрозависимые.

**Допустимо:**

- `переменная1`
- `_временноеЗначение`
- `счетчик23`

**Недопустимо:**

- `1число` (начинается с цифры)
- `myVar` (латинские буквы)
- `пере-менная` (дефис не допускается)

## 4. Литералы

| Тип литерала              | Формат                 | Примеры                               |
|:--------------------------|:-----------------------|:--------------------------------------|
| **Целые числа**           | Десятичные цифры       | `123`, `-45`, `0`                     |
| **Вещественные числа**    | Цифры с точкой         | `3.14`, `-2.5`, `0.0`                 |
| **Строки**                | В двойных кавычках     | `"Привет"`, `"Многострочная\nстрока"` |
| **Логические**            | `БАЗА` или `КРИНЖ`     | `БАЗА`, `КРИНЖ`                       |
| **Символы экранирования** | `\n`, `\t`, `\"`, `\\` | `"Строка\nс переносом"`               |

## 5. Операторы

| Тип операторов     | Операция           | Vaibik |
|:-------------------|:-------------------|:-------|
| **Унарные**        | Унарный плюс       | `+`    |
|                    | Унарный минус      | `-`    |
|                    | НЕ                 | `НЕ`   |
| **Арифметические** | Сложение           | `+`    |
|                    | Вычитание          | `-`    |
|                    | Умножение          | `*`    |
|                    | Деление            | `/`    |
|                    | Остаток от деления | `%`    |
| **Сравнения**      | Равно              | `==`   |
|                    | Не равно           | `!=`   |
|                    | Меньше             | `<`    |
|                    | Больше             | `>`    |
|                    | Меньше или равно   | `<=`   |
|                    | Больше или равно   | `>=`   |
| **Логические**     | И                  | `И`    |
|                    | ИЛИ                | `ИЛИ`  |
|                    | НЕ                 | `НЕ`   |
| **Присваивание**   | Присваивание       | `=`    |

## 6. Прочие лексемы

| Тип лексем         | Лексемы | Назначение                               | Примеры использования                 |
|--------------------|---------|------------------------------------------|---------------------------------------|
| **Разделители**    | `;`     | Завершение инструкции                    | `ВЫБРОС("Привет");`                   |
|                    | `,`     | Разделение элементов в списках           | `ВЫБРОС(а, б, в);`                    |
|                    | `.`     | Разделение целой и дробной части         | `3.14`                                |
| **Скобки**         | `()`    | Группировка выражений, параметры функций | `ЕСЛИ (условие) ТО`, `ВОЗДУХАН имя()` |
|                    | `{}`    | Блоки кода, инициализация структур       | `СБОРКА Имя { поле: Тип }`            |
|                    | `[]`    | Индексация массивов, объявление массивов | `ПАЧКА[10] ЦИФЕРКА`, `массив[5]`      |
| **Другие символы** | `:`     | Разделение идентификатора и типа         | `ФАКТ число: ЦИФЕРКА = 10`            |
|                    | `_`     | Часть идентификаторов                    | `имя_переменной`                      |

## 7. Комментарии

| Тип комментария   | Vaibik                                               | Особенности                    |
|:------------------|:-----------------------------------------------------|:-------------------------------|
| **Однострочный**  | `(ПОЯСНИТЕЛЬНАЯ-БРИГАДА: текст)`                     | До конца строки                |
| **Многострочный** | `(ПОЯСНИТЕЛЬНАЯ-БРИГАДА: ... ФИНАЛОЧКА-КОММЕНТАРИЯ)` | Может занимать несколько строк |

## 8. Встроенные типы данных

| Тип данных                   | Vaibik                     |
|:-----------------------------|:---------------------------|
| **Целые числа**              | `ЦИФЕРКА`                  |
| **Числа с плавающей точкой** | `ПОЛТОРАШКА`               |
| **Строки**                   | `ЦИТАТА`                   |
| **Логический тип**           | `РАСКЛАД (БАЗА\|КРИНЖ)`    |
| **Null-значение**            | `ПШИК`                     |
| **Объявление переменной**    | `ФАКТ имя: Тип = значение` |

## 9. Пользовательские составные типы данных

| Тип данных              | Vaibik              | Особенности                          |
|:------------------------|:--------------------|:-------------------------------------|
| **Массивы**             | `ПАЧКА[размер] Тип` | Фиксированный размер, индексация с 0 |
| **Индексация массивов** | `массив[индекс]`    | В квадратных скобках                 |

# Примеры базовых конструкций

## 1. Блоки кода

* `ПОЕХАЛИ`    Начало программы или блока кода
* `ФИНАЛОЧКА`  Конец блока кода

## 2. Ветвление (условные операторы)

| Конструкция                 | Vaibik                                                      | Особенности                                                     |
|:----------------------------|:------------------------------------------------------------|:----------------------------------------------------------------|
| **Простое условие**         | `ЕСЛИ <условие> ТО <действие>;`                             | После `ТО` может быть только один оператор (блок операторов)    |
| **Условие с альтернативой** | `ЕСЛИ <условие> ТО <действие> ИНАЧЕ <действие>;`            | После `ИНАЧЕ` может быть только один оператор (блок операторов) |
| **Множественный выбор**     | `ЕСЛИ <условие> ТО <действие> ИНАЧЕ ЕСЛИ <условие> ТО ...;` | Цепочка условий через `ИНАЧЕ ЕСЛИ`                              |

## 3. Циклы

| Тип цикла          | Vaibik                                                     | Особенности                          |
|:-------------------|:-----------------------------------------------------------|:-------------------------------------|
| **Со счётчиком**   | `ЦИКЛ (инициализация; условие; шаг) ПОЕХАЛИ ... ФИНАЛОЧКА` | Аналогичен for в C-подобных языках   |
| **С предусловием** | `ЦИКЛ (условие) ПОЕХАЛИ ... ФИНАЛОЧКА`                     | Проверяет условие до выполнения тела |

## 4. Вывод в консоль

| Действие                | Vaibik                                | Особенности                       |
|:------------------------|:--------------------------------------|:----------------------------------|
| **Вывод в консоль**     | `ВЫБРОС(выражение)`                   | Автоматический перенос строки     |
| **Ввод с консоли**      | `ВБРОС(переменная)`                   | Чтение одного значения            |
| **Множественный ввод**  | `ВБРОС(выражение1, выражение2, ...)`  | Несколько выражений через запятую |
| **Множественный вывод** | `ВЫБРОС(выражение1, выражение2, ...)` | Несколько выражений через запятую |

## 5. Пользовательские функции и процедуры

| Аспект                   | Vaibik                                 | Особенности                                  |
|:-------------------------|:---------------------------------------|:---------------------------------------------|
| **Объявление процедуры** | `ВОЗДУХАН Имя() ПОЕХАЛИ ... ФИНАЛОЧКА` | Не возвращает значение                       |
| **Объявление функции**   | `Тип Имя() ПОЕХАЛИ ... ФИНАЛОЧКА`      | Обязательное указание типа возврата          |
| **Параметры**            | `(параметр1: Тип, параметр2: Тип)`     | Типы указываются после имени через двоеточие |
| **Возврат значения**     | `ГОЙ-ДА выражение`                     | Завершает выполнение функции                 |
